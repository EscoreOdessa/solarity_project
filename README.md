EScore Energy — Автоматизатор цін для СЕС

Інструмент автоматично заповнює ціни на обладнання у кошторисі сонячної
електростанції (СЕС). Це 4-й етап робочого процесу компанії: проєктувальники
формують перелік обладнання у Google-таблиці, а скрипт підтягує до нього актуальні
ціни — з внутрішнього прайсу постачальника та з інтернету — і записує їх назад у
таблицю з посиланнями на джерела.


Статус: працює пошук та заповнення цін (цей README описує готову частину).
Розгортання «в один клік» (Streamlit) і нічний автозапуск (GitHub Actions) — у
планах, див. розділ Дорожня карта.




Що робить скрипт

price_finder.py проходить по позиціях зі вкладки «Довідник обладнання та цін»
у Google Sheets і для кожної порожньої комірки ціни:


Перевіряє прайс ETI (файл eti_04.03.2026 на Google Диску, вкладка
Price_04.03.2026). Якщо позиція знайдена — бере ціну звідти зі знижкою 25 %.
Зіставлення йде за артикулом (точно) або за назвою (fuzzy-match).
Кабельну групу не шукає в ETI. Уся кабельно-провідникова продукція та
аксесуари (кабелі, короби, лотки, канали, конектори) одразу йдуть у веб-пошук —
потрібних брендів (KOPOS, EKD, E.NEXT, DKC) у прайсі ETI немає, і fuzzy-match
на них помилявся. Керується списком ETI_SKIP_TYPE_KEYWORDS.
Шукає ціну через Serper (API Google Shopping, без браузера і без капчі).
Основне джерело — /shopping (ціна приходить готовим полем); якщо порожньо —
фолбек на органічну видачу /search.
Усереднює стійко до викидів. Відсікає явний брухт (копійки) і завищені
модифікації навколо медіани, далі за правилом інструкції відкидає мінімум і
максимум та усереднює. У примітку до комірки пишуться середнє, медіана, усі
знайдені ціни та URL-джерела.
Записує результат у таблицю: ціну, постачальника і примітку з джерелами.
Заповнюються лише порожні комірки — позиції з уже проставленою ціною
інженерів пропускаються (_is_empty_price). Цілі розділи можна виключати через
SKIP_ITEMS.


serper_test.py — допоміжний скрипт: порівнює, що віддає Serper у режимі звичайного
пошуку проти Google Shopping на двох позиціях (для вибору джерела ціни).

Налаштування (одноразово)

1. Залежності Python (у віртуальному оточенні .venv):

bashpip install gspread google-auth google-auth-oauthlib \
            google-api-python-client openpyxl rapidfuzz colorama

2. Доступ до Google (Sheets API + Drive API увімкнені у проєкті Google Cloud):


credentials.json — OAuth-ключ Desktop App з Google Cloud Console, покласти
поруч зі скриптом. Під час першого запуску відкриється браузер для входу, після
чого автоматично з'явиться token.json.


3. Ключ Serper:


Зареєструватися на serper.dev (без картки), скопіювати
API-ключ і вставити його у файл serper_api_key.txt поруч зі скриптом.



Запуск

bashsource .venv/bin/activate      # активувати оточення
python3 price_finder.py

На старті скрипт повідомить, чи знайдено ключ Serper, завантажить прайс ETI,
відкриє таблицю і пройде по позиціях, друкуючи знайдені ціни.

Безпека

Секретні файли (credentials.json, token.json, serper_api_key.txt,
service_account.json) внесені у .gitignore і ніколи не потрапляють у
репозиторій. Репозиторій — приватний.


Дорожня карта


 Перевести авторизацію на сервісний акаунт Google (робота без інтерактивного входу — для хмари і розкладу).

 GitHub Actions: автоматичне оновлення раз на 2 тижні вночі.
 Прибрати зайві залежності, упорядкувати requirements.txt.
